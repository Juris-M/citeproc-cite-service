checkCourtMap()

Check that the code element of each name-to-code pair in courtMap satifies two conditions:

  • the code is not empty; and
  • the code is registered in the courts segment of the default jurisdiction in the LRR

Collect failures in two lists corresponding to the type of failure, and issue a helpful warning for the latter and throw a helpful error for the former.
